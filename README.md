# paint-coverage-calculator-python
Python program that calculates paint consumption and cost for different purchase scenarios.

# 🎨 Paint Store Calculator (Python)

## 🇧🇷 Português

Este projeto simula um sistema de cálculo para uma loja de tintas.

O objetivo é calcular a quantidade de tinta necessária para pintar uma área informada pelo usuário, considerando diferentes formas de compra e buscando o menor custo possível.

### 📌 Regras do problema

- 1 litro de tinta cobre 6 m²
- Latas possuem 18 litros e custam R$ 80,00
- Galões possuem 3,6 litros e custam R$ 25,00
- Sempre adicionar 10% de folga na quantidade de tinta

### 🧠 Funcionalidades do sistema

O programa calcula três cenários:

1. **Apenas latas de 18L**
   - Arredonda sempre para cima

2. **Apenas galões de 3,6L**
   - Arredonda sempre para cima

3. **Mistura otimizada**
   - Adiciona 10% de margem de segurança
   - Combina latas e galões para reduzir desperdício

- Condicionais (if/else)
